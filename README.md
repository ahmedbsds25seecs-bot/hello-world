# Task Manager Application

# Overview
The Task Manager Application is a simple Python-based system for managing tasks.  
It allows users to create tasks, organize them by priority, mark them as complete, search by keywords, and delete tasks.  
This project demonstrates clean code practices, unit testing, and Responsible AI principles.

# Features
- Add tasks with a title, description, and priority (high / medium / low)
- List all tasks
- Mark tasks as complete
- Delete tasks
- Filter tasks by priority
- Search tasks by keyword
- Display tasks in a formatted output
